# STUDY FLOW API
A student productivity and task management backend system built with fastAPI and SQLAlchemy.

## Problem it solves
This FASTAPI allows students to create, manage, update and track tasks easily.

# FEATURES
- Create tasks 
- View all tasks
- Update task status
- Delete tasks
- Filter tasks by status
- Add due dates and priorities

# Tech Stach
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Uvicorn
